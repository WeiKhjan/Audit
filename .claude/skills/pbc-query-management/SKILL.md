## Query Management Workflow

1. **Raise** - Document query with W/P reference, description, date
2. **Track** - Update status (Open/Pending/Closed)
3. **Follow up** - Set deadlines, escalate unresolved items
4. **Resolve** - Record client response, close with resolution date
5. **Summarize** - Include in completion memo for unresolved items
